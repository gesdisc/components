Time Series

<terra-time-series> | TerraTimeSeries
Since 1.0 mvp

A component for visualizing time series data using the GES DISC Giovanni API.

All Data Pre-Configured 

<terra-time-series
    collection="NLDAS_FORA0125_H_2_0"
    variable="LWdown"
    start-date="01/01/2019"
    end-date="03/01/2019"
    location="33.9375,-86.9375"
></terra-time-series>

Collection and Variable Pre-Configured 

<terra-time-series
    collection="M2T1NXSLV_5_12_4"
    variable="V50M"
    start-date="05/03/2024"
    end-date="06/03/2024"
></terra-time-series>

No Pre-Configured Data 

<terra-time-series
></terra-time-series>

import TerraTimeSeries from '@nasa-terra/components/dist/react/time-series'

const App = () => <TerraTimeSeries
    collection="GPM_3IMERGHH_06"
    variable="precipitationCal"
    start-date="01/01/2019"
    end-date="09/01/2021"></TerraTimeSeries>

Importing

If you’re using the autoloader or the traditional loader, you can ignore this section. Otherwise, feel free to use any of the following snippets to cherry pick this component.

Script Import Bundler React

To import this component from the CDN using a script tag: 

<script type="module" src="https://cdn.jsdelivr.net/npm/@nasa-terra/components@0.0.57/cdn/components/time-series/time-series.js"></script>

To import this component from the CDN using a JavaScript import: 

import 'https://cdn.jsdelivr.net/npm/@nasa-terra/components@0.0.57/cdn/components/time-series/time-series.js';

To import this component using a bundler: 

import '@nasa-terra/components/dist/components/time-series/time-series.js';

To import this component as a React component:

import TerraTimeSeries from '@nasa-terra/components/dist/react/time-series';
